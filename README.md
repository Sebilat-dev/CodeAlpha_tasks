**Title**: Basic Network Sniffer Using Python

**Objective:**
To build a Python-based network sniffer that captures and analyzes network traffic packets.

**Tools Used:**
Python, Scapy, IDLE

**Procedure:**
A Python script was created using the Scapy library to capture live network packets. The program was executed to monitor traffic while browsing the web. Captured packets were analyzed to extract useful details such as source and destination IP addresses, protocols, and port numbers.

**Result:**
The program successfully captured packets and displayed useful information including source IP, destination IP, protocol type, and ports.

**Conclusion:**
This task improved my understanding of how data flows through a network and how packet sniffing works in cybersecurity.
